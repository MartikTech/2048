# 2048

A simple 2048 game in-browser made with html, css, and JS

## How it works

The game has a 4-by-4 grid, where you can use the arrow keys or the WASD keys to make the numbers and make them merge, the values that make your score go up. Your goal is to make them merge so you get 2048. Each time you move, a new time gets added, either a 2 or a 4, in a random empty space. The games either ends when you win (get 2048) or all the spots are full with no way to merge anything together. It also tracks the number of moves, time, and ofcourse the score and your highest score.
